文档展示-kibana

Kibana是一个免费且开放的用户界面

  • 对Elasticsearch 数据进行可视化
  • 在Elastic Stack 中进行导航
  • 进行各种操作
  • 跟踪查询负载
  • 下载地址：https://artifacts.elastic.co/downloads/kibana/kibana-7.8.0-windows-x86_64.zip

# 安装

  • 解压缩下载的 zip 文件
  • 修改 config/kibana.yml 文件
# 默认端口
server.port: 5602
server.host: "0.0.0.0"
# ES 服务器的地址
elasticsearch.hosts: ["http://localhost:9200"]
# 索引名
kibana.index: ".kibana"
# 支持中文
i18n.locale: "zh-CN"
